# Análisis de Señales: Dominio del Tiempo y Frecuencia

Este proyecto fue desarrollado para la materia de procesamiento de señales en la UCNL. El objetivo es analizar señales elementales utilizando la Transformada de Fourier (FFT) en Python.

## Archivos incluidos
- `script.py`: Código base para la generación y análisis de una señal senoidal.
- `Tarea.py`: Implementación adicional de otras señales.

## Instrucciones para ejecutar
1. Asegúrate de tener instalado Python.
2. Instala las librerías necesarias ejecutando:
   `pip install numpy scipy matplotlib`
3. Ejecuta el archivo que desees analizar:
   `python script.py`

## Análisis breve
Mediante el uso de la FFT, transformamos la señal del dominio del tiempo al de la frecuencia, permitiéndonos identificar su frecuencia fundamental de 5 Hz de manera precisa.
